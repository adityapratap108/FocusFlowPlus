# FocusFlow ⏳

**FocusFlow** is a sleek, minimalist productivity application designed to help users manage their time and tasks effectively using the Pomodoro Technique. Built for focus, FocusFlow provides a distraction-free environment to help you reach your goals.

[**Explore the Live Demo »**](https://adityapratap108.github.io/FocusFlow/)

---

## ✨ Features

- **Pomodoro Timer:** Fully functional timer with Work, Short Break, and Long Break intervals.
- **Task Management:** Integrated To-Do list to keep track of your objectives during each session.
- **Clean UI/UX:** A modern, responsive design that works perfectly on desktop and mobile.
- **Visual Feedback:** Real-time updates and notifications when a session is complete.
- **Persistent Data:** Uses Local Storage to keep your tasks saved even if you refresh the page.

## 🛠️ Tech Stack

- **HTML5:** Semantic structure for a modern web experience.
- **CSS3:** Custom styling with Flexbox/Grid and smooth transitions.
- **JavaScript (ES6+):** Logic for the timer, state management, and DOM manipulation.
- **GitHub Pages:** Fast and reliable hosting.

## 📝 How to Use

1. **Set your Task:** Add the tasks you want to accomplish in the task manager.
2. **Start the Flow:** Click "Start" to begin your 25-minute focus session.
3. **Short Break:** When the timer ends, take a 5-minute break to stretch.
4. **Repeat:** After 4 focus sessions, take a longer 15-minute break.
